## Dogs

### Descrição

Projeto desenvolvido durante o curso da origamid, Front-end feito todo em React utilizando suas principais propriedades, consumindo ApiDogs desenvolvida em Wordpress.

### Abrir e rodar o projeto

Após baixar o projeto, abra utilizando Visual Studio Code.
<ul>
  <li>De o comando <strong>npm i</strong> no terminal do projeto.</li>  
  <li>Após instalar as dependências faça o comando <strong>npm start</strong>.</li>
</ul>

### Tecnologias Utilizadas

<ul>
  <li>React</li>  
  <li>JavaScript</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>Api Dogs</li>
</ul>
